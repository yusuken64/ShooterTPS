Hierarchy
Demo_Scene (Dungeon_A) 설명

Main Camera --- Clear Flags 옵션을 Solid Color로 지정합니다.
                      Background 옵션을 어두운색으로 정했습니다.
                      옵션을 원하는 설정으로 변경할 수 있습니다.

Light --- Directional Light_Mixed만 Mixed 모드입니다 나머지 라이트는 Baked 모드입니다.
            라이트맵을 굽기 위해서는 라이트가 모두 켜져 있어야 합니다.
            사용후 필요없는 라이트는 숨겨두기 또는 삭제 하시면 됩니다.
            텍스처 사이즈 1024 한 장 나오도록 설정을 하였습니다.
            Lighting - Scene - Lightmap Resolution 을 10.2 로 하였습니다. 퀄리티를 올리고 싶으면 수치를 더올리시면 됩니다.
            프리팹 - Mesh Renderer - Scale In Lightmap 옵션 조절 해도됩니다.  (Na_Cliff - Scale In Lightmap 을 0.4 로 하였습니다.)
            Fbx파일에 uv2채널이 있습니다. (라이트맵용 입니다.)  

BG --- Effect, Natures, Props 폴더로 나누었습니다.
         Effect --- 이펙트 프리팹, 이펙트가 포함된 프랍 프리팹이 있습니다.
         Props 폴더에는 개별 폴더를 만들었습니다.
         Natures 폴더에는 개별 폴더를 만들었습니다.
                  
Navi --- 큐브를 이용해서 네비게이션을 만들었습니다. (Unity 2018 - GameObject - 3D Object - Cube)
           사용후 숨겨두기 또는 삭제 하시면 됩니다.
