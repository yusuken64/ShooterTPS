Hierarchy
Demo_Scene (Dungeon_A) description

Main Camera --- The Clear Flags option to Solid Color.
                      The Background option is set to dark.
                      You can change the options to the desired settings.

Light --- Directional Light_Mixed is only in blended mode, the rest is baked mode.
            You have to turn on all the lights to baked the light map.
            You can hide or delete unnecessary lights after use.
            Texture size 1024 is set to appear one.
            Lighting-Scene-Lightmap Resolution is set to 10.2 If you want to increase the quality, increase the number.
            You can also adjust the Prefab - Mesh Render - Scale In Lightmap option. (Na_Cliff-Scale In Lightmap is set to 0.4)
            Fbx file has uv2 channel. (For light maps.)  

BG --- Divided into Effect, Natures, and Props folder.
         Effect --- There is an effect prefab, and a prop prefab with effects.
         I created an individual folder in the Props folder.
         I created an individual folder in the Natures folder.
                  
Navi --- I created a navigation using the cube. (Unity 2018 - GameObject - 3D Object - Cube)
           You can hide or delete it after use.

Module description

The cliff was placed using Na_Cliff_01, Na_Cliff_02, Na_Cliff_03. The height is 5m.
I created a floor using Unity Terrain.
You can also use the plane (Te_Ground_01). (You need a program that connects the tile Textures. Also sold in the Unity Asset Store.)
You can create terrain using Unity Terrain. There is a cliff tile texture. (Te_Ground_04.png)
The Pr_Point_01 prefab also contains an effect prefab.
I created a tree prefab using Leaf.FBX and Trunk.FBX.
   Na_Tree_01 : Na_Leaf_01 + Na_Trunk_01
   Na_Tree_02 : Na_Leaf_01 + Na_Trunk_02
   Na_Tree_03 : Na_Leaf_02 + Na_Trunk_01
   Na_Tree_04 : Na_Leaf_02 + Na_Trunk_02
